# 💳 Wallet-Bivix

In this project, a virtual wallet will be developed.

## 🚀 Technologies Used

- **Ruby**
- **Sinatra** (as the web framework)
- **JavaScript**
- **HTML5**
- **CSS3**
- **Gemfile** for managing Ruby dependencies

## 📌 Features

✅ User registration and login

✅ Account creation

✅ Card and transaction management
